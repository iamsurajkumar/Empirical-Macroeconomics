# Repository Structure

This repository has been reorganized into the following categories to improve navigability:

## 1. Machine Learning & Transformers
**Directory:** `Machine_Learning/`
Contains files related to double descent phenomena, transformer models, and ML benchmarks.
- `DoubleDescent.jl`
- `test_double_descent.jl`
- `run_experiments.jl`
- `nk_causal_transformer_irf_diagnostics.ipynb`
- `nk_cowles_varx_benchmark.ipynb`
- `double_descent_tutorial.tex`
- `README_DOUBLE_DESCENT.md`
- `tensor2tensor/`

## 2. Economic Papers & Replications
**Directory:** `Papers/`
Contains code and data related to specific economic papers.
- `Borovickova-Shimer (2025)/`
- `Hopenhayn (1992) Model/`

## 3. Time Series & Econometrics
**Directory:** `Time_Series/`
Contains time series analysis methods and models.
- `VARs/`
- `Local Projections/`

## 4. Analysis
**Directory:** `Analysis/`
Contains general analysis notebooks.
- `stock_market_analysis.ipynb`

## 5. Root Files
- `Project.toml`: Julia project dependencies.
- `INSTALLATION.md`: Installation instructions.
- `structure.md`: This file.
