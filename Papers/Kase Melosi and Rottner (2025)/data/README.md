# Data Directory

This directory stores observed data for model estimation.

## Structure

```
data/
├── README.md           # This file
├── synthetic/          # Synthetic datasets for testing
├── us_macro/           # U.S. macroeconomic data
└── other/              # Other empirical datasets
```

## Data Files

### Synthetic Data
- Generated by particle filter workflow
- For testing estimation procedures
- Files: `synthetic_*.csv` or `*.jld2`

### U.S. Macroeconomic Data
- Quarterly data (FRED, BEA, BLS)
- Observable variables:
  - Output growth (annualized quarterly)
  - Inflation (annualized quarterly)
  - Nominal interest rate (annualized)
- Files: `us_data_*.csv`

### Data Format

Preferred formats:
- **CSV**: For small datasets, easy inspection
- **JLD2**: For large datasets, faster I/O in Julia
- **HDF5**: For cross-platform compatibility (Julia ↔ Python)

## Usage

```julia
using CSV, DataFrames

# Load observed data
data = CSV.read("data/us_macro/fred_quarterly.csv", DataFrame)

# For particle filter
y_obs = Matrix(data[:, [:output_growth, :inflation, :interest_rate]])
```

## Notes

- Large data files should not be committed to git
- Add data files to `.gitignore` if > 10 MB
- Document data sources and preprocessing steps
